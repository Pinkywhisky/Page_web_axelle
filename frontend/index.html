Bienvenue au Club des Pattes

Un espace pour gérer les membres du club, les profils de vos animaux et les accès administrateurs.

Suivi des membres Profils animaux Rôles client / admin

Pour les membres 🐾

  • Créez votre compte via le bouton “S'inscrire” en haut à droite.
  • Connectez-vous avec votre e-mail et votre mot de passe.
  • Votre rôle par défaut est client.

Pour l'équipe du club 🐕‍🦺

  • Les comptes admin peuvent être configurés dans le fichier clients.json.
  • L’API Flask permet de gérer les membres (création, modification, suppression).
  • Les données sont stockées côté serveur dans un fichier JSON.

💡 Astuce : créez d'abord un compte client, puis changez son rôle en admin dans clients.json pour vous donner les droits d’admin.
