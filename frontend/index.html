Bienvenue au Club des Pattes

Un espace pour suivre vos inscriptions, les profils de vos animaux et la gestion du club.

Suivi des membres Profils animaux Vue admin dédiée

Pour les membres 🐾

  • Créez votre compte via le bouton “S'inscrire” en haut à droite.
  • Connectez-vous pour retrouver votre profil et les informations de votre animal.
  • Vos données restent côté club et ne sont pas partagées.

Pour l'équipe du club 🐕‍🦺

  • Connectez-vous avec un compte admin pour accéder à la vue de gestion.
  • Consultez la liste des inscrits, gérez les rôles et supprimez des fiches si nécessaire.
  • Les données sont stockées dans un fichier JSON côté serveur.

💡 Astuce : si vous n'avez pas encore de compte, commencez par créer un profil via le bouton “S'inscrire”. Une fois connecté, votre vue s'adaptera automatiquement à votre rôle (client ou admin).
